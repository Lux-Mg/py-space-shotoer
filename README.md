# Space Shooter

A classic 2D space shooter game built with Python and Pygame. Defeat waves of enemies, dodge their lasers, and survive as long as possible.

Игра в жанре космический шутер на Python с использованием Pygame. Уничтожайте волны врагов, уклоняйтесь от их лазеров и выживайте как можно дольше.

## Features

- 4 ship types (yellow player + red/green/blue enemies)
- Increasing difficulty across levels
- Health and lives system
- Laser shooting with cooldown
- Collision detection using pixel masks
- Pixel-art assets

## Tech Stack

- **Language:** Python 3
- **Library:** Pygame
- **Platform:** Cross-platform (Windows, Linux, macOS)

## Controls

| Key | Action |
|---|---|
| `A` / Left arrow | Move left |
| `D` / Right arrow | Move right |
| `W` / Up arrow | Move up |
| `S` / Down arrow | Move down |
| `Space` | Shoot laser |

## Installation

```bash
git clone https://github.com/Lux-Mg/py-space-shotoer.git
cd py-space-shotoer/Space\ Shooter
pip install pygame
python main.py
```
